Die Verbrauchsanzeige

Wissen, was im Inneren Ihres Tesla passiert. durch das Power Consumption Feature haben Sie den Energierverbrauch Ihres Fahrzeugs stets im Blick.
Lassen Sie sich Verbrauchsdetails als Tabelle oder Diagramm anzeigen und erleben Sie Energie-Management auf einem neuen Level.

Die Power Consumption Anzeige informiert Sie über den absoluten Engergie-Verbrauch seit Ihrer letzten Ladung, sowie über die Daten Ihrer einzelnen Fahrten.
Sowohl die zurückgelegte Distanz, als auch die verbrauchte Gesamt-Energie wird Ihnen unter der DEFAULT-Anzeige zur Verfügung gestellt.
Die Ansicht des Verbrauchs-Diagrams stellt Ihnen diese Daten auf Wunsch grafisch dar.
das Power Consumption Feature passt sich optisch an die aüßeren Lichtverhältnisse an.
